Como saber o que foi modificado no seu projeto:

        git diff


Como saber qual o nome do arquivo que foi alterado:

        git diff --name-only


como saber o que foi aterado em um arquivo específico:

        git diff [nomearquivo]


como voltar as alterações de um arquivo da mesma branch antes do commit:

        git checkout HEAD -- [nomeArquivo]
